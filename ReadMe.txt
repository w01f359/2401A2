Purpose: To simulate a function call stack inorder to help users track the execution of their programs

Header files: stdio.h, string.h, stdlib.h, stackDefs.h

Source files: main.c, loop.c, stack.c

To compile: gcc main.c loop.c stack.c -o runme

To run: ./runme

Program Instructions:

1. Enter the number of elements
2. Enter each element(number) one at a time
3. Choose either '1' to sum iteratively, or '2' to sum recursively
